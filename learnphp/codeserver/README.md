# PHP

![](https://upload.wikimedia.org/wikipedia/commons/thumb/2/27/PHP-logo.svg/440px-PHP-logo.svg.png)

PHP originally stood for Personal Home Page.  It is a core part of this system because it can allow us to run code on a server from the browser without any command line activity on just about any web server.  PHP comes standard on even free web hosting, and "just works" unlike other server side web frameworks which all require command line package installations.  

People like to look down on PHP but this is based on ignorance and snobbery.  We do not need to use this to "write software" in any complex way.  We use PHP to do the absolute bare minimum required to interact with the server side file system.  This means we need to be able to:

 - copy files
 - delete files
 - save to a file 
 - load from a file
 - parse strings
 - work with JSON objects and arrays
 - run python for data acquisition from Arduino
 - list files, list directories
 - create a new directory
