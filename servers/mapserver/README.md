#  Map Server

Make a map, replicate a map, delete a map.  Maps are a format of arranging things geometrically in a web browser. Those things can be text and/or images, and can be global web links or local links to other maps.  See Book Of Geometron for details, available free online at [www.trashrobot.org](https://www.trashrobot.org/).
